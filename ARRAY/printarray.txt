Pattern: Single Pass Iterative Pattern

This pattern is used because:

You need to perform an operation on every element (printing each).

The task does not require searching, sorting, or skipping elements.

The operation is O(n) â€” perfect for small, direct operations.


Type	Complexity	Reason
Time	O(n)	You visit each element once
Space	O(1)	No extra memory except loop variable
